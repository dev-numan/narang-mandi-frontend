# SEO Checklist — Narang Mandi News (narangmandi.com)

Goal: rank on Google for **narang mandi**, **narangmandi**, **narangmandi.com**,
**Narang Mandi News** and the Urdu name **نارنگ منڈی نیوز**.

The code/on-page work is **done** (see bottom). This file is the **non-code work**
— the part that actually earns rankings, and only you can do it.

---

## ✅ Already done in the code (foundation)
- Working `robots.txt` and dynamic `sitemap.xml` (auto-updates as you publish).
- Pre-rendered HTML for crawlers (real headlines, text, internal links — not a blank page).
- Per-page `<title>`, meta description, Open Graph + Twitter tags.
- Schema.org JSON-LD: `WebSite`, `NewsMediaOrganization`, and `NewsArticle` on articles.
- Site name set to **"Narang Mandi"** (schema `name` + `og:site_name`).
- Latin brand "Narang Mandi News" in titles/meta so English-letter searches match.
- `hreflang="ur-PK"` + `lang="ur"` (targets Urdu / Pakistan).
- Image cache headers; nginx upload limit raised to 10 MB.

You don't need to touch these. Focus on the steps below.

---

## 1. Google Search Console (do first — weekly)
- [ ] Verify the property `narangmandi.com` (DNS TXT).
- [ ] Submit the sitemap: `sitemap.xml` (Sitemaps → Add). Status should read **Success**.
- [ ] **URL Inspection → Request Indexing** for the homepage.
- [ ] Request Indexing for your 3–5 most important articles.
- [ ] Each week: request indexing for new important articles; check **Pages** for coverage growth.
- [ ] Watch **Performance** for which queries bring clicks (confirm "narang mandi" shows up).

## 2. Backlinks & consistent brand name (the biggest lever)
Google trusts you're the real "Narang Mandi" brand when other places say so and link to you.
- [ ] **Facebook Page** named exactly **Narang Mandi News** → put `https://narangmandi.com` in the About/website field, and link it in posts.
- [ ] **WhatsApp Channel** / groups → pin the website link.
- [ ] **YouTube channel** (if any) → website link in the channel "About" + video descriptions.
- [ ] **Instagram / X (Twitter)** bio → website link.
- [ ] Share **every** new article link on all of the above (each share is a signal).
- [ ] Ask 2–3 local businesses / bloggers / friends' sites to link to narangmandi.com.
- [ ] List the site in any Pakistan / local news directories you can find.

> Use the **exact same name** ("Narang Mandi News") and the **same URL**
> (`https://narangmandi.com`) everywhere — consistency is what Google rewards.

## 3. Google Business Profile (for local "narang mandi" searches)
- [ ] Create a **Google Business Profile** for "Narang Mandi News" (category: News Service / Media).
- [ ] Add the website `https://narangmandi.com`, location (Narang Mandi), and contact.
- [ ] This helps you appear in local/map searches for the town.

## 4. Publish consistently (freshness)
- [ ] Post local news **regularly** (aim for several times a week). Active news sites get crawled more.
- [ ] Put the real topic/keywords in each **article title** (that's what ranks the article).
- [ ] Write a **1–2 line excerpt** for every article (it becomes the search snippet).
- [ ] Add a **cover image** to every article (used for social/Google previews).
- [ ] Link between related articles inside your content when it makes sense.

## 5. Monthly review
- [ ] Search `site:narangmandi.com` — count should grow as more pages get indexed.
- [ ] Search `narang mandi news` and `narangmandi` — confirm you appear (and move up).
- [ ] In Search Console, check for **coverage errors** and fix any pages not indexed.
- [ ] Re-run the [Rich Results Test](https://search.google.com/test/rich-results) after big changes.
